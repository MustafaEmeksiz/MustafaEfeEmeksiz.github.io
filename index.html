Sales Insights in PowerBI

This project analyzes over 200,000 rows of sales data from AtliQ India, covering four years of transactions across multiple cities. I performed data cleaning and transformation using Power Query, standardized currencies to INR, and removed invalid or duplicate entries. Data integrity was verified using SQL, and a star schema model was implemented for optimal performance. Key measures were created using DAX, and insights were presented through a Power BI dashboard with three focused pages. The final product enables interactive exploration of sales performance, regional trends, and time-based patterns.

Information System Computer Hardware Faults

In this thesis project, I built a full data pipeline to analyze historical hardware fault data from company computers. I created and inserted synthetic data using Python and structured it in a MySQL database. The data was cleaned, organized, and analyzed to identify patterns and trends in hardware failures. For visualization, I designed Excel-based dashboards to support insight generation and reporting. This project demonstrates end-to-end data handling, from creation and transformation to analysis and dashboarding, with a focus on reliability and clarity in operational data.
